# BasicDungeonRL

This is an open source roguelike based on the Basic Fantasy Rpg.

### Prerequisites
* Python 3, tested on 3.6 but should work with other versions.
* bearlibterminal
* clubsandwich -- Wheel included, in case other versions don't work.
* sortedcontainers
* tdl
